# RSI + AO + Stochastic Entry System (TradingView)

Two-script TradingView indicator system:
- Panel script: RSI / Stoch / AO + divergences + setups
- Chart overlay script: BUY / SELL arrows

## Files
- `pine/zaBor_RSI_AO_Stoch_indicator.pine`
- `pine/zaBor_RSI_AO_Stoch_chart.pine`
- `User_Manual.md`

## Install (TradingView)
1. Open TradingView → Pine Editor
2. New → Blank indicator
3. Paste `zaBor_RSI_AO_Stoch_indicator.pine` → Save → Add to chart
4. Repeat for `zaBor_RSI_AO_Stoch_chart.pine` → Add to chart

## Alerts
Create alerts on the panel script conditions (BUY / SELL), recommended: Once Per Bar Close.
