# zaBor RSI + AO + Stochastic Entry System (TradingView)

TradingView entry indicator for structured BUY / SELL signals.

## Features
- RSI regular & hidden divergences
- RSI momentum confirmation (v0.2)
- Optional Awesome Oscillator filter
- Stochastic trigger with reversal continuation
- Cooldown anti-spam protection

## Files
- `pine/zaBor_RSI_AO_Stoch_indicator.pine`
- `pine/zaBor_RSI_AO_Stoch_chart.pine`
- `User_Manual.md`
- `CHANGELOG.md`

## Installation
1. Open TradingView → Pine Editor
2. Paste `zaBor_RSI_AO_Stoch_indicator.pine`
3. Save → Add to chart
4. Repeat for chart overlay script

## Timeframes
- 1H — balanced frequency
- 4H — high-confidence signals
